# PBMS-Server

图片管理系统服务端

## feature

- [x] 图片水印(文字、图片)
- [x] 图片压缩(比例)
- [x] 上传图片限制(大小、类型)
- [x] 图片存储(多级目录)
- [x] HTTPS支持
